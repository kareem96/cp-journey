# Sliding Window

Lanjutan LANGSUNG dari Two Pointer (bukan pattern baru total) — bedanya di
sini "jendela" (window) antara 2 pointer yang membesar/mengecil sesuai
kondisi, bukan cuma saling mendekat dari 2 ujung.

## Rencana soal

**Easy:**
- [ ] Best Time to Buy and Sell Stock (#121)
- [ ] Contains Duplicate II (#219)

**Medium (setelah Easy lancar):**
- [ ] Longest Substring Without Repeating Characters (#3)
- [ ] Longest Repeating Character Replacement (#424)
